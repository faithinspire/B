## 🐛 TROUBLESHOOTING

### Issue: Button not dragging
**Solution**: 
1. Make sure you're clicking on the button itself
2. Try on a different browser
3. Clear cache and refresh

### Issue: Chat window too small
**Solution**:
1. Hard refresh (Ctrl+Shift+R)
2. Clear browser cache
3. Check browser zoom (should be 100%)

### Issue: Map modal not opening
**Solution**:
1. Click the "🗺️ View Map" or "🗺️ Open Map" button
2. Check browser console for errors
3. Try on a different browser

### Issue: AI not responding
**Solution**:
1. Check internet connection
2. Check if `/api/ai/chat` endpoint is working
3. Try refreshing the page
4. Check browser console for errors
