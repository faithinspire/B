## 📊 SYSTEM ARCHITECTURE

```

    ↓
/forgot-password page
    ↓
Enter email address
    ↓
POST /api/auth/forgot-password
    ↓
Brevo SMTP API
    ↓
Email sent to user
    ↓
User receives email
    ↓
User clicks reset link
    ↓
/reset-password page
    ↓
User enters new password
    ↓

    ↓
Password updated
    ↓
Redirect to /login
    ↓
User logs in with new password
    ↓
Dashboard access
```
